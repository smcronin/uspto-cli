## Bugs Found

- `--type` help text lists wrong code for Design patents (`DSN` shown, `DES` works)
- `--download csv` ignores range filters and structured filters
- `--filed-after` + `--granted` can cause silent 404
- `--filed-within` can fail silently when combined with other flags
- `petition search --sort` fails without text query
- `ptab decisions-for` appears not to return institution decisions
- Older patent citations may not parse from legacy grant XML formats
