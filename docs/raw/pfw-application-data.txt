Application Data

The Application Data section contains key bibliographic information found on the front page of granted patents and published applications, as well as additional data not on the front page. To view the full Swagger documentation, visit the Swagger UI.

This dataset includes published patent applications and issued patent data filed after January 1, 2001.

GET
https://api.uspto.gov/api/v1/patent/applications/{applicationNumberText}/meta-data
e.g. Returns application data for application number
Note: API Key is required. Obtain an API key.

If you have any questions or errors, please visit the Support page for assistance.
