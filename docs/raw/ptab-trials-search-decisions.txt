All public PTAB Trial decisions (i.e., briefing filed by parties and decisions by PTAB) are available since September 2012. Other details for using the Search endpoint: If you do not specify which attributes you would like to see in the response, it returns all data attributes. All search syntaxes are applicable to this endpoint, meaning any number of combinations is possible.
