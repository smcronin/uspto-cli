Search

Conduct a search of all patent application bibliographic/front page and patent relevant data fields.

Use this endpoint if you are interested in searching across multiple patents or applications. For example, you may want to search for applications that have a specific "Utility" or "Design" application type.

The other endpoints (such as Application Data, Continuity, etc.) should be used when you're using a specific application number and want to get details about a specific application.

Note: Other details for using the Search endpoint:

- The Search API has a maximum of 10,000 results. If you need more, please refine your search.
- Search results are sorted by relevance by default, but you can sort by any field.

All search syntaxes are applicable to this endpoint, meaning any number of combinations is possible. For more information on the query syntaxes, please visit the Open Data Portal API page or use the Swagger UI.

You can download the JSON Schema file to parse the JSON responses returned by the APIs.

GET
https://api.uspto.gov/api/v1/patent/applications/search
Returns a list of all patents and applications that match your search term.
Note: API Key is required. Obtain an API key.

POST
See Swagger documentation (https://data.uspto.gov/swagger/index.html)
Returns a list of all patents and applications that match your search term.
Note: API Key is required. Obtain an API key.

If you have any questions or errors, please visit the Support page for assistance.

=== 400 - Bad request ===
Bad request.
The server was unable to understand the request due to malformed syntax or invalid parameters. This error is usually caused by incorrect query parameters or a missing required parameter.
Update your syntax or parameters and try again.
If you have any questions or additional errors, please visit the Support section for assistance.

=== 403 - Access denied ===
Access denied.
Access to the requested patent or application is forbidden. There are many reasons this could be the case, including that the application is not yet published or that the application is restricted.
If you have any questions or additional errors, please visit the Support section for assistance.

=== 404 - Not found error ===
Not found error.
The server was unable to find the requested resource or URL. This code is commonly returned when you are looking for an application that does not exist.
If you have any questions or additional errors, please visit the Support section for assistance.

=== 413 - Request Entity Too Large ===
Request Entity Too Large.
The server is refusing to process a request because the request payload is larger than the server is willing or able to process.
If you have any questions or additional errors, please visit the Support section for assistance.

=== 500 - Internal server error ===
Internal server error.
An unexpected condition has occurred on the server, and the server was unable to fulfill the request.
If you have any questions or additional errors, please visit the Support section for assistance.
