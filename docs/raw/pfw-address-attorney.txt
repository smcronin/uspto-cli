Use this endpoint when you want address and attorney/agent information related to a specific patent application whose application number you know. You can test APIs right away in Swagger UI . This dataset includes published patent applications and issued patent data filed after January 1, 2001. This data is refreshed daily. GET https://api.uspto.gov/api/v1/patent/applications/{applicationNumberText}/attorney Returns attorney/agent data for supplied application number. Note: API Key is required. Obtain an API key.
