Use this endpoint when you want documents data for a specific trial whose Trial number you know. All public documents filed in PTAB Trial Documents (i.e., briefing filed by parties and decisions by PTAB) are available since September 2012. You can also test the API right away in Swagger UI . GET https://api.uspto.gov/api/v1/patent/trials/{trialNumber}/documents/ e.g. https://api.uspto.gov/api/v1/patent/trials/IPR2025-01412/documents/ Returns Trial Document data for a Trial number. Note: API Key is required. Obtain an API key. If you have any questions or errors, please visit the Support page for assistance. Response200400403404413500Successful request.
