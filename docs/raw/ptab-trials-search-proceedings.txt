All public PTAB Trial proceedings (i.e., briefing filed by parties and decisions by PTAB) are available since September 2012. Other details for using the Search endpoint: If you do not specify which data attributes you would like to see in the response, it returns all data attributes. All search syntaxes are applicable to this endpoint, meaning any number of combinations is possible. Some example requests are below, but for more information, view the Simplified Syntax documentation that is linked in the API Syntax Examples page. You can also test the API right away in Swagger UI . You can download the JSON Schema file to parse the JSON responses returned by the APIs. GET https://api.uspto.gov/api/v1/patent/trials/proceedings/search Returns a list of all Trial Proceedings that match your search term. Note: API Key is required. Obtain an API key.
