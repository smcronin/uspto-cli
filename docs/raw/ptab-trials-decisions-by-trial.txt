Use this endpoint when you want Decisions data for a specific trial whose Trial number you know. All public documents filed in PTAB Trial Decisions (i.e., briefing filed by parties and decisions by PTAB) are available since September 2012. You can also test the API right away in Swagger UI . GET https://api.uspto.gov/api/v1/patent/trials/{trialNumber}/decisions e.g. https://api.uspto.gov/api/v1/patent/trials/IPR2025-01222/decisions Returns Trial Decisions data for a Trial number.
