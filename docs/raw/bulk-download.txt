Use this endpoint when you want to download bulk data sets. Note: You are currently limited to 20 downloads of the same file per year for each API Key to avoid overtaxing the service. On the 21st request, you will get an error message with HTTP status response 429. GET https://api.uspto.gov/api/v1/datasets/products/files/{productIdentifier}/{fileName} Returns a binary stream of data with the content of the requested file name. Note: API Key is required. Obtain an API key. If you have any questions or errors, please visit the Support page for assistance.
