This dataset includes published patent applications and issued patent data filed after January 1, 2001. This data is refreshed daily. GET https://api.uspto.gov/api/v1/patent/applications/{applicationNumberText}/foreign-priority Returns foreign priority data for supplied application number. Note: API Key is required.
