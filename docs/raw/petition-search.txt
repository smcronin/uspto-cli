Use this endpoint if you are interested in searching across multiple patents or applications and their corresponding final petition decisions. Note: Petition decisions data are incrementally added to ODP on a monthly basis starting with data from 2022 and later. Review our release notes page to find what data is currently available. Other details for using the Search endpoint: If you do not specify which attributes you would like to see in the response related to the search term(s), it returns all data attributes. All search syntaxes are applicable to this endpoint, meaning any number of combinations is possible. Some example requests are below, but for more information, view the Simplified Syntax documentation that is linked in the API Syntax Examples page. You can also test the API right away in Swagger UI . You can download the JSON Schema file to parse the JSON responses returned by the APIs. GET https://api.uspto.gov/api/v1/petition/decisions/search Returns a list of all petition decisions that match your search term. Note: API Key is required. Obtain an API key.
